# YouTube 트렌드 분석 — 경제 브리핑 시스템 활용 기획 문서

> 작성일: 2026-05-31
> 목적: 유튜브 트렌드 데이터를 경제 시황 텔레그램 봇에 통합하기 위한 기획 참고 자료

---

## 1. 왜 유튜브 트렌드가 필요한가

### 1-1. 기존 데이터 소스의 한계

현재 텔레그램 봇에서 활용하는 뉴스 API(The Guardian, BigKinds)는 **"무슨 일이 일어났는가"** 라는 팩트를 제공한다.
그러나 주식 시장은 팩트만으로 움직이지 않는다. **"사람들이 어떻게 반응하고 있는가"** 라는 심리(센티먼트) 데이터가 함께 있어야 시장 방향성 예측이 가능하다.

```
뉴스 API         → 팩트 (What happened)
유튜브 트렌드    → 심리 (How people are reacting)
```

이 두 가지를 결합하면 뉴스만으로는 보이지 않는 **시장 참여자의 군중 심리**를 파악할 수 있다.

### 1-2. 유튜브가 개미 심리 지표인 이유

국내 주식 시장 참여자 구조:
- 기관·외국인 → 블룸버그 터미널, 리서치 보고서, 전문 뉴스레터
- 개인 투자자(개미) → **유튜브를 가장 먼저 본다**

개미 투자자의 정보 탐색 루트:
```
이슈 발생
    ↓
유튜브 검색 ("삼성전자 왜 떨어져", "지금 팔아야 해")
    ↓ (수 시간 ~ 1일 후)
포털 뉴스 댓글 / 커뮤니티
    ↓ (1~2일 후)
주식 매매 행동으로 연결
```

즉 **유튜브 트렌드는 개미 수급의 선행 지표**다.
특정 종목·섹터 관련 유튜브 조회수·참여율이 급등하면, 이후 개인 투자자 자금 유입 또는 유출로 이어질 가능성이 높다.

## 2. 데이터 소스 역할 정의

### 전체 데이터 구조

```
┌─────────────────────────────────────────────────────────┐
│                   경제 브리핑 시스템                      │
├─────────────┬──────────────┬──────────────┬─────────────┤
│ The Guardian│   BigKinds   │  유튜브 API   │  금융 데이터 │
│  해외 뉴스  │  국내 뉴스   │  대중 심리    │  (추후 추가) │
│   (팩트)    │   (팩트)     │ (센티먼트)    │  주가/환율   │
└─────────────┴──────────────┴──────────────┴─────────────┘
                          ↓
                     Gemini API
                   (종합 분석·요약)
                          ↓
               텔레그램 하루 3회 전송
```

### 각 데이터 소스별 역할 비교

| 데이터 소스 | 측정하는 것 | 신뢰도 | 실시간성 |
|---|---|---|---|
| The Guardian | 해외 경제 사건 | 높음 | 수십 분 |
| BigKinds | 국내 언론 보도 | 높음 | 수십 분 |
| 유튜브 트렌드 | 대중 관심도·감정 | 중간 | 수 시간 지연 |
| 금융 데이터 API | 가격·지표 | 높음 | 실시간 |

## 3. 유튜브 트렌드 활용 방법

### 3-1. mostPopular 차트 방식 (현재 vibe-06 구현)

**방식:** YouTube `chart=mostPopular&regionCode=KR` 엔드포인트 사용
**기준:** YouTube 자체 알고리즘 (공식 "인기 급상승" 탭과 동일)

YouTube의 트렌드 선정 기준 (공식 발표 기준):
- 조회수 절대값이 아닌 **상승 속도**
- 조회수 발생 **출처** (유튜브 내부 검색 vs 외부 유입)
- 업로드 후 경과 시간 대비 **성과**
- 클릭률(CTR), 평균 시청 지속 시간

**한계:**
- 수 시간 지연 → 장중 활용 부적합
- 카테고리 제한 (교육·IT 등 일부 카테고리 데이터 부족)
- 브리핑에서 보조 지표로만 활용 권장

### 3-2. 키워드 추적 방식 (실제 활용에 적합)

**방식:** 특정 경제 키워드로 최근 업로드 영상의 참여율 변화를 매일 추적

```
추적 키워드 예시:
- 거시경제: "코스피", "환율", "금리", "인플레이션"
- 섹터: "반도체", "2차전지", "바이오", "AI"
- 개별 종목: "삼성전자", "SK하이닉스" 등
```

**측정 방식:**
```
매일 동일 키워드로 최근 7일 이내 영상 검색
    ↓
총 조회수 합산 / 평균 참여율 계산
    ↓
전일 대비 변화율 산출
    ↓
임계값 초과 시 "관심도 급등" 신호 발생
```

**신호 해석 예시:**
```
"삼성전자" 키워드 참여율 전일 대비 +150% 급등
→ 개미 투자자 관심 급증
→ 단기 개인 수급 변화 가능성 → 브리핑에 포함

"2차전지 팔아야해" 관련 영상 급등
→ 패닉 셀링 심리 확산 신호
→ 해당 섹터 단기 하방 압력 예상
```

### 3-3. 참여율의 중요성

조회수만 보면 안 되는 이유:

```
A 영상: 조회수 100만 / 좋아요 1,000 / 댓글 100 → 참여율 0.11%
B 영상: 조회수  10만 / 좋아요 5,000 / 댓글 500 → 참여율 5.5%
```

- A: 알고리즘 추천으로 노출됐지만 관심 없이 흘려봄
- B: 진짜 관심 있는 사람들이 적극적으로 반응함

**투자 심리 측면에서 B가 훨씬 강한 신호다.**
참여율 = (좋아요 + 댓글) / 조회수 × 100

## 4. 하루 3회 브리핑과의 연동 설계

### 4-1. 7시 — 모닝 브리핑

```
[전일 해외 시장 현황]        ← The Guardian API
[오늘 국내 시장 진단]        ← BigKinds API
[전일 유튜브 경제 트렌드]    ← YouTube API (신규 추가)
```

**유튜브 섹션 내용 예시:**
```
📊 어제 급상승한 경제 키워드
• "환율 1400원" 관련 영상 참여율 +230% (개인 투자자 우려 확산)
• "반도체 실적" 관련 콘텐츠 급증 (삼성·하이닉스 어닝 시즌 관심)
• "배당주 추천" 영상 꾸준한 상승세 (안전자산 선호 심리)

→ 오늘 외국인/개인 수급 참고 시 활용
```

### 4-2. 12시 — 오전장 브리핑

유튜브 데이터는 수 시간 지연으로 **장중 활용 부적합** → 생략 권장
실시간 뉴스와 실제 주가 데이터로만 구성하는 것이 정확도 높음

### 4-3. 18시 — 저녁 브리핑

```
[당일 국내 시장 결산]        ← BigKinds + 금융 데이터
[금일 유튜브 반응 요약]      ← YouTube API (신규 추가)
[내일 시장 영향 분석]        ← Gemini 종합
```

**유튜브 섹션 내용 예시:**
```
📱 오늘 유튜브 반응
• 오전 장 하락 후 "지금 저가 매수 타이밍" 영상 폭증
  → 개인 투자자 반등 기대 심리 강함, 내일 오전 단기 매수세 유입 가능성
• "코스피 2400 붕괴" 관련 콘텐츠 공포 분위기
  → 심리적 지지선 논의 활발, 변동성 지속 예상
```

## 5. 기술 구현 방안

### 5-1. 현재 프론트엔드 프로토타입 (vibe-06)

- YouTube Data API v3 기반
- 키워드 검색 + 카테고리별 트렌드 확인
- 참여율 계산 및 Chart.js 시각화
- **역할:** 수동 조회 및 아이디어 검증용

### 5-2. 텔레그램 봇 자동화 구현 (백엔드 필요)

**필요 스택:**
```
Python 백엔드 (Contabo 서버)
├── youtube-data-api (google-api-python-client)
├── 키워드별 참여율 추적 로직
├── 전일 대비 변화율 계산
├── Gemini API (트렌드 해석 및 요약)
└── Telegram Bot API (전송)
```

**자동화 흐름:**
```
Cron (매일 06:30 실행)
    ↓
사전 정의 키워드 목록으로 YouTube 검색
    ↓
최근 7일 영상 참여율 집계
    ↓
전일 대비 변화율 계산
    ↓
임계값(예: +100%) 초과 키워드 추출
    ↓
Gemini API로 시황 해석 문장 생성
    ↓
07:00 텔레그램 전송
```

### 5-3. 추적 키워드 관리

키워드는 고정값과 동적값으로 분리 관리:

```python
# 고정 키워드 (항상 추적)
FIXED_KEYWORDS = [
    "코스피", "코스닥", "환율", "금리",
    "반도체", "2차전지", "바이오",
    "삼성전자", "SK하이닉스"
]

# 동적 키워드 (뉴스 API에서 오늘의 핫 키워드 자동 추출)
# → BigKinds/Guardian에서 언급 빈도 높은 단어 자동 추가
```

## 6. 한계 및 주의사항

| 항목 | 내용 |
|---|---|
| 데이터 지연 | mostPopular는 수 시간 지연 → 장중 활용 불가 |
| 노이즈 | 자극적 제목의 유튜브 특성상 공포·과열 신호가 과장될 수 있음 |
| 인과관계 | 유튜브 트렌드 ≠ 주가 상승 보장. 어디까지나 보조 지표 |
| API 할당량 | YouTube Data API 무료: 10,000 units/일. 키워드당 약 100 units 소비 |
| 해석 주의 | 유튜브 트렌드는 개미 심리 지표. 기관·외국인 동향은 별도 파악 필요 |

```
├── BigKinds 뉴스 핫 키워드 → 유튜브 자동 연동
├── 전일 대비 변화율 알림 임계값 튜닝
└── 섹터별 유튜브 심리 지수 시각화 대시보드
```

## 8. 요약

> 유튜브 트렌드는 **개인 투자자(개미) 심리의 선행 지표**다.
> 뉴스 API가 팩트를 제공한다면, 유튜브 트렌드는 그 팩트에 대한 **대중의 감정적 반응**을 측정한다.
> 단독으로 매매 판단에 쓰는 것은 위험하지만, 뉴스·가격 데이터와 결합해 보조 지표로 활용하면 브리핑의 완성도를 높일 수 있다.
